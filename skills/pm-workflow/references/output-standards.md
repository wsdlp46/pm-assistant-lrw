# 输出标准

## 通用规范

- 中文专业沟通，输出对齐标准规范
- 功能描述控制在 120 字以内，阿拉伯数字分点
- 交付物达到可直接向客户展示的水平

## PRD 必含项

1. 角色与端侧定义（平台类产品前置）
2. 功能列表（按模块组织，阿拉伯数字分点）
3. Out of Scope 章节（明确不在范围内的内容）
4. 非功能需求（性能、安全、合规）
5. 验收标准

## 方案设计必含项

1. 场景描述
2. 痛点分析
3. 方案概述
4. 关键决策点表格（决策→备选→理由→影响）
5. 报价/排期（如适用）

## 原型设计必含项

1. 信息架构图
2. 关键页面原型
3. 交互说明（弹窗层级、跳转逻辑）
4. 遵循 B 端原型 6 条准则
